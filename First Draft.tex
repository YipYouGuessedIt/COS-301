\section{Architecture Constraints}

\subsection{Technologies that must be used : }

\begin{itemize}
\item{HTML : }For handling the structure of the webpages returned by the PHP.
\item{CSS : }For styling the webpages.
\item{PHP : }For handling the creation of webpages and handling client side requests including input.
\item{MySQL : }For handling the databases used in the application.
\item{CAPTCHA : }For preventing spam attacks.
\end{itemize}

\subsection{Technologies that may be used : }
\begin{itemize}
\item{PERL : }For handling the creation of webpages and handling client side requests instead of PHP.
\item{Javascript : }For handling the creation of webpages and handling client side requests in conjunction with PHP.
\item{AJAX : }For facilitating communication between forms.

\end{itemize}
